Software Projects

Symbolism

Symbolism is a C# library which implements automatic simplification of algebraic expresions. Computer algebra systems perform automatic simplification to convert expressions into a canonical form. In many computer algebra systems, it is a fundamental step around which the rest of the system is built. Symbolism can be called from F#. The implementation algorithms are ported from my other computer algebra project, MPL.

Experimental Music Compiler

Experimental Music Compiler is a Windows Phone app (versions 7 & 8) which generates algorithmic music in the bytebeat genre. I wanted to experiment with bytebeat music on my Windows Phone but the JavaScript apps on the web didn't seem to play well with the WP browser. I wrote this app in C# and XAML to experiment with bytebeat songs directly on the phone.

MPL

MPL is a library for computer algebra written in R6RS Scheme.

Box2d-Lite in Scheme

Box2d is a popular C++ physics engine. Box2D-Lite is a lightweight version of Box2D. I implemented a port of Box2D-lite to Scheme. These are not bindings to the C++ library. This is a complete, from scratch rewrite in 100% Scheme. One notable technique employed in this project was the implementation of macros which implement dot notation for accessing structure elements as well as calling methods on the structures.

Continuous integration system for Factor

I implemented a continuous integration system for the Factor programming language project team. This system was written in Factor itself. The system is still being used by the Factor team.

Factor module system

I designed and prototyped the module system for the Factor programming language. The design was accepted and implemented by the main developer.

Dataflow combinators for Factor

I contributed the cleave, spread, and apply family of data flow combinators to the Factor programming language. More information on these combinators can be found in the official Factor documentation.

Xlib bindings for Factor

I implemented Factor bindings to the X Window System client library (xlib). These bindings were used to provide the Factor user interface on Linux and BSD systems. Developed the event dispatch system which was employed by the user-interface backend.
